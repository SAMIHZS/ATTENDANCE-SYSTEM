# Attendance Management System

A modern, institutional attendance management system built for high-scale academic tracking. Designed with a mobile-first philosophy, rich aesthetics, and robust administrative reporting.

## 🚀 Quick Start
For detailed installation, database migrations, and demo personas, see the **[RUNBOOK.md](./RUNBOOK.md)**.

## ✨ Core Features

### 👨‍🏫 Teacher Flow
- **Live Class Detection**: Automatically identifies the current session based on the timetable.
- **Focused Roll Call**: A high-efficiency UI card model for marking attendance student-by-student.
- **Attendance Ledger**: History of submitted sessions with summary stats.

### 🎓 Student Flow
- **Visual Analytics**: Interactive dashboard showing overall and subject-wise attendance percentages.
- **Attendance History**: Searchable log of every class session with status indicators.
- **Shortage Alerts**: Visual cues for subjects falling below the required 75% threshold.

### 🛡️ Admin Suite
- **Institutional Health**: Dashboard metrics for total students, teachers, and active classes.
- **Attendance Explorer**: Complete audit log of all sessions with the ability to override records.
- **Reporting & Exports**: Filterable reports by Class, Subject, and Date with one-click CSV/Excel export.
- **Timetable Management**: Manage weekly schedules for different classes and teachers.

## 🛠️ Technology Stack
- **Frontend**: Vite, React, TanStack Query, Tailwind CSS.
- **Backend**: Node.js, Express, TypeScript.
- **Database**: Supabase (PostgreSQL), Postgres functions (RPCs).
- **Design**: Google Material Design (M3) inspiration + Glassmorphism.
